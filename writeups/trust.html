Máquina: Trust (Fuzzing y PrivEsc)

02/09/2024 El Pingüino de Mario

Este desafío se centró en la fase de reconocimiento activo utilizando fuzzing en una aplicación web, seguido por una escalada de privilegios simple.

1. Fuzzing y Descubrimiento

Iniciamos el proceso usando Ffuf para buscar directorios y archivos ocultos en la web. El objetivo era encontrar una ruta que expusiera información sensible.

ffuf -w /usr/share/wordlists/dirbuster/directory-list-2.3-small.txt -u http://172.17.0.3/FUZZ -e .php,.html

Descubrimos el directorio /uploads que permite la subida de archivos, lo cual es un vector de ataque potencial.

Consejo: El fuzzing es crucial. Nunca te fíes de lo que ves a simple vista en una web.

2. Escalada de Privilegios

Usamos un exploit conocido para el servidor web y conseguimos una shell de bajo privilegio. Para la escalada, aprovechamos un archivo de configuración mal protegido que tenía credenciales hardcodeadas, lo que nos permitió convertir la shell a root.

cat /var/www/html/config.php
su root

Tras obtener las credenciales, el acceso como administrador fue inmediato. Máquina completada.
